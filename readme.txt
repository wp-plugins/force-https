Forces every single page/post to go over HTTPS.


== Description ==
This plugin adds a permanent 301 cacheable redirect to go your entire site over HTTPS.
Make sure you have SSL installed on your domain.

== Installation ==
This section describes how to install the plugin and get it working.

1. Make sure you have SSL installed on your domain.
2. Unzip and Upload the force-https folder to your plugin directory.
3. Activate it. and everything will be working and all your pages will be saved over secure HTTPS.
4. `define(\'FORCE_SSL_ADMIN\', true);` in your wp-config.php

== Frequently Asked Questions ==
What should i do after activating the plugin?
Nothing! You just need to install and active the plugin. After activating the plugin it will automatically be working and will server you entire website over SSL.
